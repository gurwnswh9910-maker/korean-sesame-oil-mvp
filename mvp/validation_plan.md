# 한국식 방앗간 참기름 MVP 검증 계획

## 검증 목표

일본 소비자가 `한국 방앗간에서 갓 짠 듯한 향`에 실제 행동을 보이는지 확인한다. 칭찬, 좋아요, 설문 의향이 아니라 신청, 회신, 샘플 구매 요청, 과거 구매 행동을 본다.

## MVP 형태

- 일본어 정적 랜딩 페이지
- CTA: Notion 입하 알림/선예약 폼. 현재 공개 폼은 `Submission` 1줄 입력 방식이며, 랜딩에서 복붙용 응답 메모를 제공한다.
- 가격 앵커: 100ml 1,480엔, 3병 세트 3,980엔
- 제품 상태: 정식 판매 전 수요 검증, 수입/표시 확인 후 소량 입하 안내
- 채널: 신오쿠보/K-food 홈쿡/한국여행/향미 오일 커뮤니티 수동 게시
- 응답 저장: Notion database `韓国式しぼりたてごま油 MVP 検証`, X 공개 응답 URL 수동 수집, GitHub Issue fallback

## 2주 일정

| Day | 작업 | 산출물 |

## KPI

| 지표 | 정의 | 성공 기준 | 중단 기준 |
|---|---|---:|---:|
| CTA 클릭률 | 랜딩 방문 대비 폼/메일 버튼 클릭 | 3% 이상 | 1% 미만 |
| 신청 전환율 | 랜딩 방문 대비 폼 제출 또는 공개 응답 | 1.5% 이상 | 0.5% 미만 |
| 가격 수용률 | 신청자 중 제시 가격에도 구매 의향 회신 | 30% 이상 | 10% 미만 |
| 과거 행동률 | 인터뷰 중 최근 한국 참기름 구매/탐색 경험 보유 | 30% 이상 | 10% 이하 |
| 샘플 구매 요청 | 실제 결제/입금 가능성을 묻는 사람 | 5명 이상 | 1명 이하 |

## 폼 질문

현재 Notion Form에는 아래 항목을 1줄 응답으로 붙여넣게 한다. 다문항 폼 도구로 전환하면 각 항목을 독립 필드로 분리한다.

1. 최근 6개월 안에 산 참기름 브랜드와 구매처는?
2. 한국 여행 중 시장/방앗간/식당에서 참기름 향이 기억난 경험이 있는가?
3. 주로 어떤 요리에 참기름을 쓰는가?
4. 100ml 1,480엔이면 입하 알림을 받을 의향이 있는가?
5. 3병 3,980엔 선예약이면 누구와 나눠 쓸 생각이 있는가?
6. 배송비 포함/별도 중 어떤 조건이면 구매가 멈추는가?
7. 유입 채널 또는 댓글 가능한 공개 채널

## 고객 인터뷰 질문

- 마지막으로 참기름을 산 때는 언제였고, 어디서 샀나요?
- 그때 왜 그 브랜드를 골랐나요?
- 한국 참기름을 일본에서 다시 찾으려 한 적이 있나요?
- 지금 쓰는 참기름에서 아쉬운 점이 있다면 실제로 무엇을 바꿔봤나요?
- 가격이 100ml 1,480엔이면 어떤 요리에만 쓰게 될까요?
- 선물용이면 몇 병 단위가 자연스러운가요?

## 공개 게시 전 체크

- 폼 URL 확정
- 개인정보 수집 문구 확인
- 판매 확정 표현 제거
- 수입신고/라벨링 전에는 결제 받지 않음
- 배송 가능 국가, 예상 입하 시점, 환불/취소 안내 문구 정리
