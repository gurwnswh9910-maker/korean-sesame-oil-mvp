# note 제품 외 콘텐츠 게시 패킷

작성일: 2026-06-27

상태: 초안 준비, 실제 게시 전 최신 로그인 세션과 중복 게시 리스크 확인 필요

## 목적

기존 note 글은 제품/입하 안내에 가까웠다. 두 번째 note를 낸다면 제품 판매글이 아니라, 타겟 밀집구역을 찾기 위한 콘텐츠형 질문글로 낸다.

검증할 핵심 문제:

- 한국여행에서 산 참기름을 일본 집에서 끝까지 썼는가
- 신오쿠보/한국식품점에서 이미 어떤 참기름을 사고 있는가
- 큰 병은 향이 약해지는 문제가 있는가
- 제조일, 강한 향, 100ml 소량, 재구매 편의가 바꿀 이유가 되는가

## 게시 후 기록할 것

- 게시 URL
- 게시 시각
- 24h/72h 전체뷰, 댓글, 스키
- Notion 응답 수
- note 댓글 중 최근 구매처/브랜드/용량/가격/향 보존/전환 조건 포함 건수

## 금지선

- 기존 제3자 note 글에 홍보 댓글을 달지 않는다.
- 결제, 예약금, 배송지, 전화번호, 이메일을 받지 않는다.
- `한국산이면 일본 제품보다 낫다`처럼 대체재를 깎는 문장을 쓰지 않는다.
