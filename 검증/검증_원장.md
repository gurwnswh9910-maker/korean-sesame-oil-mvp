## 왜 완료가 아닌가

MVP와 게시물은 만들어졌고 공개되었지만, 아직 일본 소비자 또는 관련 커뮤니티에서 온 응답이 없다. 따라서 현재 할 수 있는 말은 "게시와 측정 경로 구축 완료"이지 "시장 검증 완료"가 아니다.

## AI스러움 적대적 검증

| 의심 지점 | 문제 | 조치 | 남은 리스크 |
|---|---|---|---|
| GitHub Pages URL | 소비자에게 개발 실험처럼 보임 | Vercel clean URL을 canonical 공개면으로 전환하고 note 본문 링크도 Vercel로 교체 | GitHub Pages fallback에는 기록용으로 남음 |
| `.html` 경로 | 급조한 정적 파일처럼 보임 | `/note`, `/share`, `/field` route를 Vercel에서 검증 | GitHub Pages fallback에는 `.html`이 남음 |
| GitHub 공개 답변 CTA | 참기름 소비자에게 부자연스러움 | Vercel 소비자 화면에서 GitHub CTA 제거 | 내부 자동 집계용 fallback은 남김 |
| 1줄 Notion Form | 폼으로는 어색하고 수동 복붙 느낌이 있음 | 30초 응답 메모로 정직하게 설명 | 반응이 약하면 Tally/Google Forms/Typeform류로 교체 필요 |
| 모바일 note에서 QR/외부폼 전환 | 폰에서 QR을 찍으라는 CTA는 다른 기기가 필요하고, 외부 폼 direct 전환도 이탈을 만든다 | `/answer-note`를 note 댓글/Notion 폼/복사용 메모 선택 허브로 변경 | note 원문은 같은 링크를 유지하므로 새 허브 배포 검증 필요 |
| 제품 사진/브랜드 실체 | 실제 병/라벨/방앗간 증거가 아직 없음 | 현재는 판매 전 수요 검증이라고 명시 | 다음 단계에서 실제 샘플/제조일/병 사진 없으면 신뢰 부족 |
| note 계정 신규성 | 첫 글/새 계정 표시가 신뢰를 낮출 수 있음 | 사용자 허용 하에 note 게시 완료, Vercel 보조 페이지 제공 | 계정 이력 부족은 당장 해결 불가 |
