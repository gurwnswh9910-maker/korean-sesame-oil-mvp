## 결정해야 하는 질문

다음 액션은 `6차 note를 올릴까`가 아니라 아래 질문에 답하게 해야 한다.

1. note가 타겟에게 도달하지 못했는가?
2. 도달했지만 응답을 남길 만큼 문제가 선명하지 않았는가?
3. 응답 경로가 귀찮거나 신뢰가 약했는가?
4. 기존 대체재로 이미 충분해서 상품성 자체가 약한가?

## KPI 프레임

### Primary KPI

`strong problem-fit response count`

아래 8개 중 5개 이상을 포함한 원본 응답만 strong으로 센다.

1. 최근 구매처 또는 현재 보고 있는 매장/사이트
2. 후보 상품명, 브랜드, 店名 중 하나
3. 용량 또는 가격 기억
4. 1병을 다 쓰는 데 걸리는 기간
5. 개봉 후 향 저하, 마지막 향 부족, 보관 불안, 큰 병 부담 중 하나
6. Kadoya 5g使い切りパック, Kadoya/Kuki/Ottogi/Kim-san/신오쿠보 搾りたて 같은 기존 대체재와 비교
7. 제조일/搾った日/시식/차광병/1~2개월 사용량/재구매 편의 중 필요한 증거
8. 100ml 1,480円에 대한 조건부 긍정 또는 명확한 거부 이유

2026-06-28T12:13+09:00부터 이 기준은 `scripts/summarize_validation_signals.py`의 `problem_fit_score`로 자동 집계한다. summary에는 `Strong problem-fit responses`와 `Highest problem-fit score`가 표시된다.

2026-06-28T12:17+09:00부터 note dashboard 값은 `scripts/record_note_dashboard_snapshot.py`로 `experiments/note_dashboard_snapshots.csv`에 기록한다. summary에는 최신 snapshot의 total views/comments/likes와 `24h action gate`가 표시된다.

### Driver Metrics

| 지표 | 정의 | 결정에 쓰는 법 |
|---|---|---|
| qualified views | note/Vercel에서 타겟 콘텐츠가 실제로 본 조회 | 조회가 낮으면 유통 실패로 보고 게시 수보다 채널 전환을 검토 |
| answer starts | note 댓글 클릭, `/answer-note`, `/quick-answer`, Notion 폼 진입 또는 복사 메모 사용 | 조회는 있는데 응답 시작이 없으면 CTA/신뢰/마찰 문제 |
| purchase-context mentions | 구매처/브랜드/용량·가격을 포함한 응답 비율 | 호감과 실제 수요를 분리 |
| use-up burden mentions | 1병 사용 기간, 큰 병 부담, 개봉 후 향 저하가 있는 응답 수 | 100ml 소량 가격 방어 가능성 확인 |
| substitute-gap mentions | Kadoya 5g使い切りパック, Ottogi/Kuki/Kadoya/Kim-san 등으로 충분하지 않다는 이유가 있는 응답 수 | 기존 대체재 대비 남는 문제 확인 |
| price-conditioned positives | 100ml 1,480円을 조건부로 받아들이는 응답 수 | 소량 생산/프리미엄 가격 후보 판단 |

### Guardrails

- 조회수, 스키, 좋아요, 공유만으로 수요를 세지 않는다.
- 판매 확정, 예약금, 배송지, 전화번호, 이메일을 받지 않는다.
- 제3자 글에 홍보 댓글을 달지 않는다.
- `한국식이면 더 좋다` 같은 국적 우월 메시지를 쓰지 않는다.
- 기존 대체재를 모르는 가격 긍정은 약한 신호로 둔다.

## 2026-06-28T17:27+09:00 post-gate 외부 채널 보정

상세 실행 패킷은 `mvp/post_gate_external_channel_packet_20260628.md`다.

- note 1~5 합산 views가 30 미만이면 `문안 실패`보다 `유통 실패`로 보는 기준은 유지한다. 실제 24h gate는 views 36/응답 0이라 6차 note를 게시했지만, 게시 직후에도 응답은 0이다.
- X/Threads는 링크 포함 MVP 홍보보다 URL 없는 공개 질문 1회를 우선한다. 질문은 `ごま油を炒め油ではなく、料理の最後に香りを足す油として使うことがありますか？`다.
- Konest는 한국 시장/제유소/搾りたてごま油 후기가 있어 타겟 밀도는 높지만, 커뮤니티 규칙 리스크가 있어 즉시 게시하지 않는다. 허용되더라도 링크/가격/판매 없이 여행 경험 질문만 한다.
- Instagram/Lemon8/Threads 신오쿠보 맥락은 관찰/레퍼런스 우선이다. 이미지/영상 자산 없이 새 계정으로 게시하면 봇처럼 보일 가능성이 높다.
- 오프라인 신오쿠보/한국식품점 인터뷰는 여전히 가장 직접적인 고의도 검증이다. 계정 없이도 구매처/브랜드/용량/가격/사용 기간을 얻을 수 있기 때문이다.
