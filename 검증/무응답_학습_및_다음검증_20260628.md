## 무응답에서 배운 것

1. note 단독 게시만으로는 아직 유통이 약하다.
2. `한국식 방앗간 참기름`이라는 상품명보다, 소비자가 이미 겪는 문제 언어가 더 필요하다.
3. 현재 가장 유효한 문제 후보는 `큰 병을 다 쓰기 전 향이 약해짐`, `지금 사는 브랜드/매장과 비교할 기준 없음`, `밥 한 그릇에 마지막 향을 더하고 싶음`이다.
4. `韓国料理が好き`, `韓国海苔が好き`, `TKGが好き`는 넓지만 약하다. 참기름 단품 구매 이유와 연결되지 않으면 수요가 아니다.
5. 다음 live 액션은 note를 하나 더 올리는 것보다, 더 강한 문제 문장 또는 더 높은 의도 채널을 먼저 정해야 한다.

## KPI 재설계

### Primary KPI

`strong problem-fit response count`

아래 7개 중 4개 이상을 포함한 원본 응답만 센다.

1. 최근 구매처 또는 탐색처
2. 브랜드/매장/시장명
3. 용량 또는 가격 기억
4. 실제 사용 요리
5. 개봉 후 향 저하 또는 마지막 향 부족 경험
6. 일본에서 다시 살 결정요인
7. 100ml 1,480円에 대한 긍정/조건부 반응

### Driver Metrics

| 지표 | 정의 | 쓰임 |
|---|---|---|
| qualified exposure | note/Vercel/외부 콘텐츠에서 목표 타겟이 볼 가능성이 있는 노출 | 유통 실패와 메시지 실패 분리 |
| answer attempt | `/quick-answer`, `/answer-note`, note 댓글, Notion form으로 이동 또는 원본 답변 | 전환 마찰 확인 |
| purchase-context mention rate | 응답 중 구매처/브랜드/용량·가격이 있는 비율 | 호감 반응과 실제 수요 분리 |
| aroma-retention mention rate | 응답 중 향 보존/개봉 후 약해짐이 있는 비율 | 100ml 소량 가격 방어 가능성 확인 |
| shelf-candidate mention rate | 응답 중 지금 보거나 최근 본 후보 상품/용량/가격이 있는 비율 | 매대 앞 선택 피로와 단순 관심 분리 |

### Guardrails

- 조회수, 스키, 좋아요, 공유만으로 수요를 세지 않는다.
- 판매 확정, 결제, 배송지, 전화번호, 이메일 수집은 하지 않는다.
- 제3자 글에 홍보 댓글을 달지 않는다.
- `한국산이면 무조건 낫다`는 카피를 쓰지 않는다.

## 03:07 향 손실 언어 보강

`/aroma-loss-goma` 배포 후 공개 자료를 다시 확인했다. 결론은 `큰 병은 다 쓰기 전에 향이 약해진다`는 문제 언어는 유효하지만, `100ml 소량`만으로 1,480円을 방어하기는 약하다는 것이다.

| 보강 신호 | 의미 | 다음 질문 |
|---|---|---|
| Kadoya 공식 보관 Q&A | 개봉 후 풍미 저하와 적정 사이즈 선택은 소비자에게 설명 가능한 문제 | 지금 병을 연 지 얼마나 됐고, 1~2개월 안에 다 쓰는가 |
| Kuki 향 차별화/진한 향 상품 | 일본 기존 브랜드도 이미 향을 판다 | 한국식이라는 말보다 제조일/압착/마지막 향 보존 증거가 필요한가 |
| Yahoo!知恵袋의 개봉 후 기한/큰 병 회피 언어 | 소비자 언어는 `酸化`보다 `匂い`, `大きいボトルは買わない`에 가깝다 | 남은 병의 향이 약해졌는지 쉽게 묻는다 |
| 韓国広場 110ml 한국 참기름 | 저가 소량 한국 대체재가 이미 있다 | 1,480円은 소량이 아니라 신선도/제조일/압착 증거로 방어해야 한다 |

상세 리서치는 `research/05_aroma_retention_review_language_20260628.md`를 본다. 이 보강도 수요 응답은 아니므로 전체 응답자 수는 0명 그대로다.
