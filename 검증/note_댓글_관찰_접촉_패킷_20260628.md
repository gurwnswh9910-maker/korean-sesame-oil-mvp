# note 댓글 관찰/접촉 패킷

작성 시각: 2026-06-28T14:20+09:00

## 목적

note 1~5차 글이 공개됐지만 2026-06-28T14:00+09:00 기준 실제 응답은 0명이다. 사용자가 2026-06-28T14:19 전후로 이 프로젝트에 한해 note는 자유롭게 쓰고, 외부 공개 글은 자유롭게 관찰해도 된다고 허용했다.

따라서 다음 실험은 새 note를 하나 더 올리는 것이 아니라, 이미 관련 맥락을 쓰고 있는 note 글에 아주 소수의 맥락형 댓글을 남겨 응답 씨앗을 만드는 것이다.

이 문서는 댓글 게시 완료 기록이 아니다. 2026-06-28T14:20+09:00 현재 인앱 브라우저는 note 로그아웃 상태이고, 일반 Chrome 프로세스에는 CDP 원격 디버깅 포트가 없어 Codex가 댓글을 직접 전송하지 못했다.

## 권한과 금지선

허용:

- note 계정은 이 프로젝트에 한해 Codex가 자유롭게 게시/수정/댓글 작성 가능.
- 외부 공개 글, 공개 검색 결과, 공개 커뮤니티 글은 관찰 가능.
- note 댓글은 글을 실제로 읽고 맥락에 맞을 때만 작성.

금지:

- 외부 X, Threads, Instagram, LINE, Yahoo!知恵袋, Konest 등에 새 계정 로그인 또는 게시를 임의로 하지 않는다.
- note 외부 채널에는 별도 계정이 있어도 새 게시/댓글/DM을 하지 않는다.
- 링크를 댓글에 붙이지 않는다. 특히 MVP/Vercel/Notion 링크를 바로 남기지 않는다.
- 판매, 예약, 결제, 배송지, 이메일, 전화번호, DM 유도 금지.
- 같은 문장 반복 댓글 금지.

운영 제한:

- 하루 note 댓글 3개 이하부터 시작.
- 답글이 오면 먼저 공개 원본 URL을 남기고, `scripts/record_public_social_response.py`로 응답 원문을 기록한다.
- 응답이 구매처/브랜드/용량·가격/1병 사용 기간/기존 대체재/필요한 증거 중 하나도 없으면 strong response로 세지 않는다.

## 게시할 댓글 후보

### 1. Ottogi パンアッカンごま油 리뷰

대상: https://note.com/kohanaatkitchen/n/n6c024194cdfa

의도: 이미 일본에서 살 수 있는 한국식 소병 대체재로 충분한지 확인한다.

댓글:

```text
オットギのパンアッカンごま油、加熱せずに食べたいという視点がまさに知りたかったです。
もし差し支えなければ、開封後どれくらいの期間で使い切ると香りが良いまま感じられますか？
5gの使い切りや110ml小瓶で十分なのか、少し大きめでも大丈夫なのか迷っています。
```

강한 응답 조건:

- 사용 기간
- Ottogi 110ml로 충분한지/부족한지
- 향이 남는 요리 장면

### 2. 한국 로컬 제유소/비빔밥 글

대상: https://note.com/kanshoku_shuto/n/n783033624017

의도: `한국 현지 방앗간 향`이 일본 집밥에서 어떤 사용 장면으로 이어지는지 확인한다.

댓글:

```text
亀尾の製油所の香りの描写、すごく具体的で読んでいて気になりました。
韓国の搾りたてごま油って、日本に持ち帰った後はどんな料理で一番香りが生きると思われますか？
ビビンバ以外だと、冷奴やナムルみたいに最後に少しかける使い方でも違いを感じますか。
```

강한 응답 조건:

- 향이 가장 살아나는 요리/타이밍
- 일본에서 다시 살 이유
- 비빔밥 외 반복 사용 장면

### 3. 신오쿠보 소울시장/ナムル 글

강한 응답 조건:

- 가격/용량 또는 향 보존 판단

## 실행 블로커

2026-06-28T14:20+09:00 확인:

- 인앱 브라우저 `https://note.com/`는 `ログイン`, `会員登録`이 보이는 로그아웃 상태.
- 일반 Chrome 프로세스에는 `--remote-debugging-port`가 없다.
- 2026-06-28T14:37+09:00 공개 API 기준 후보 3개는 모두 `published`, `can_comment=true`다.
- 따라서 Codex가 현재 세션에서 note 댓글을 직접 전송할 수 없다.

다음에 댓글을 실제로 게시하려면 아래 중 하나가 필요하다.

1. 사용자가 인앱 브라우저에서 note에 로그인.
2. 사용자가 CDP 포트가 열린 Chrome에서 note 로그인 완료.
3. 사용자가 댓글 문안을 직접 게시하고, Codex는 원본 URL/댓글 URL을 받아 기록.

## 전용 계정 판단

당장 필요한 전용 계정은 없다.

- note: 이미 프로젝트용 계정이 있고 허용됨. 실제 댓글/게시의 1순위.
- X: 유통이 계속 약하면 나중에 전용 계정 1개를 만들 가치가 있다. 지금 필수는 아니다.
- Threads/Instagram: 사진/생활감 없이는 봇처럼 보여 보류.
- Yahoo!知恵袋: 계정이 있어도 홍보/유도성 질문 리스크가 커 관찰 전용.
- LINEオープンチャット: 개인 전화번호/계정 리스크가 커 관찰도 보수적으로.
